# List UI

A little website that parses a list and provides an interface to view it.

## List Format Example

This website expects a text file that looks something like this:

```
# Heading 1

List Item 1
    Sub Item 1
    Sub Item 2
List Item 2
List Item 3


# Heading 2

List Item 4
    Sub Item 3
List Item 5
List Item 6

```
